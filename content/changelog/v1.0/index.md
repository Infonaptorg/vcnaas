- **Page Builder** — Build landing pages with drag-and-drop blocks
- **Block System** — Hero, Features, CTA, Testimonials, Stats, and more
- **Tailwind CSS 4** — Modern utility-first styling with dark mode support
- **Preact Components** — Fast, lightweight client-side interactivity
- **SEO Optimized** — Automatic sitemaps, RSS feeds, and structured data
- **Multi-language** — 35+ language packs out of the box
